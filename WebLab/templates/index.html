How I Built This Page

Installation Steps

1. Installed Python and Flask.
2. Set up the Flask project structure.
3. Created a simple route in Flask (`app.py`).
4. Designed this page using an HTML template and CSS.

Challenges Faced

- Setting up Flask on a VPS for the first time.
- Debugging CSS issues for proper page layout.

Solutions

- Followed Flask's official documentation.
- Used a VPS (DigitalOcean) to deploy the Flask app.
- Debugged CSS using browser developer tools.
